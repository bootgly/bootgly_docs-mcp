MCP server extension for the [Bootgly PHP Framework](https://github.com/bootgly/bootgly) documentation.

Search and access Bootgly docs directly from **GitHub Copilot Chat** in VS Code.

## Features

- Search Bootgly documentation pages
- Get detailed information about Bootgly APIs, classes, and modules
- Access framework guides and examples

## Usage

After installing, the MCP server tools are automatically available in GitHub Copilot Chat.

Open the Chat view (`Ctrl+Alt+I`) and ask questions about Bootgly:

```
How do I create an HTTP server with Bootgly?
```

```
What is the I2P architecture in Bootgly?
```

## Requirements

- VS Code 1.99.0 or later
- GitHub Copilot Chat extension
